# `flat` CLI

Get the `flat` CLI from [sevenval/flat-cli](https://github.com/sevenval/flat-cli).

Start the local FLAT server with:

```sh
$ flat start
```

Run

```sh
$ flat -h
```

for all available commands and options of the FLAT cli.
